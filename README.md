# FOOT-BOT
A versatile chatbot built with Python that focuses on football (soccer) while also being adaptable for general use.

🔹 Football Features:

Tactical analysis and match discussions

Player profiles and career breakdowns

Live stats integration for games

🔹 General Features:

Casual conversation for everyday use

Extendable design to add custom topics

This project is designed to showcase how AI-driven chatbots can merge domain-specific expertise (football) with general conversational abilities..
